# Context Engineering Skills
- **Category:** Context management
- **What it does:** A collection of skills for managing an agent's context window—summarizing, pruning, and structuring information to keep long tasks within budget.
- **Repo:** https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering
- **Compatibility:** Claude Code ✅ | Codex ✅ | OpenCode ✅
- **Install:**
  - Claude Code: `npx skills add muratcankoylan/Agent-Skills-for-Context-Engineering`
  - Codex / OpenCode / others: `npx skills add muratcankoylan/Agent-Skills-for-Context-Engineering`
- **Verify:** Ask the agent to list skills; context-engineering skills appear.
- **When to use / skip:** Use for long-context or token-constrained work needing active context management; skip for short, simple tasks.
- **Stars:** ~16.9k (as of 2026-07)
